# my_first_react_project


## Overview

**What this project is:**

The idea here is to learn the basic structure of a project and fill in each block little by little.

**What it does today:**

Show a "Hello World" screen.

**What you want to learn from it:**

- **Components:** Understand how to break a website down into small, reusable building blocks (like buttons, headers, and cards).
- **JSX:** Learn this syntax extension that blends HTML with JavaScript to describe what the UI should look like.
- **Props (Properties):** Discover how to pass data from one component to another (e.g., changing a button's text or color through parameters).
- **State (useState):** Understand how to make a component "remember" things and react to user interactions (e.g., a click counter or toggling a menu open/closed).
- **Dynamic Rendering:** Learn how to display lists of data and use conditions to show or hide elements (e.g., "if the user is logged in, show the profile").
